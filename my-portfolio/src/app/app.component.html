Projects

Some of my recent works at a glance.

JPAW Web App

CSU Monterey Bay
May 2022

Student-led recreation of the Justice Partner Access Website (JPAW). This project was directed by the Superior Court of California in Monterey Bay and offers a search portal for public court data (Cases, Hering Information, Traffic Citations, etc...). The objective for this project was to create an reimagined "updated" User Interface for JPAW. This updated version was a Progressive Web Application built withg Angular. At it's core, the project prioritized consistent site performance/experience across different devices, finding ways to remove “technical debt” acquired during the previous development cycle (i.e. old dev practices), and greatly diminish the cost of maintaining three separate applications (web, iOS, Android) by consolidating the experience into one web application.
